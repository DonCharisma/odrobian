CLI Tool "insta-ORGA"

What's here?

Installation

It's all done simply through "insta-ORGA" command line tool. First, you have to do this before running this utility and it's very important, make sure you have the latest version everytime.


	odroid:$ sudo -s
	odroid:# apt-get update && apt-get install io-cli
Now, go ahead and check it out through: 

	odroid:# insta-ORGA --help
Which is the same material you can find in this example here. So, if you want to install with my default settings optimized for your specific ODROID platform you should definitely install the whole thing with either this or that. 

	Automatic Installation:
	odroid:# insta-ORGA --defaults
	odroid:# insta-ORGA --enable-autostart

	Install ORGA-X11:
	odroid:# insta-ORGA --mali=x11 --kodi=true --sound=pulse --lightdm=true

	Install ORGA-Fbdev:
	odroid:# insta-ORGA --mali=fbdev --kodi=true --sound=alsa --lightdm=false

	Customization Wizard:
	odroid:# insta-ORGA (No Flags)

	Defaults = "Our recommended default settings for your current platform with Mali-X11"
	Autostart = "Auto-Booting to EmulationStation directly"

Do not bother forcing ALSA over pulse if you are using the X11 version, I have prepared my scripts to automatically decide when to use ALSA for emulation and when to prefer PULSE for desktop use. It seems that ODROID XU4 can't give sound properly from Kodi while ALSA is activated however, don't worry it will enable required optimizations for it by default!

ODROBIAN "insta-ORGA" Utility

After using it to install your ORGA Entertainment system you can benefit more from this, even if you are a regular ODROBIAN user.

Enable/Disable Lightdm

	odroid:# insta-ORGA --lightdm=true
	odroid:# insta-ORGA --lightdm=false
Upgrade Cores/Emulators/Software

	odroid:# insta-ORGA --upgrade
Update/Restore Emulation Configs

	odroid:# insta-ORGA --update-cfg
Autostart EmulationStation

	odroid:# insta-ORGA --enable-autostart
Revert Autostart Back to Default

	odroid:# insta-ORGA --disable-autostart
Completely Remove Kodi

	odroid:# insta-ORGA --kodi=false
Force ALSA (Completely Remove Pulse)

	odroid:# insta-ORGA --sound=alsa
Get Pulse Back in Service (Associated with ALSA)

	odroid:# insta-ORGA --sound=pulse
Switch World (From ORGA X11 to ORGA Fbdev)

	odroid:# insta-ORGA --mali=fbdev --kodi=true --sound=alsa --lightdm=false
Switch World (From ORGA Fbdev to ORGA X11)

	odroid:# insta-ORGA --mali=x11 --kodi=true --sound=pulse --lightdm=true
ORGA is currently available for the "s805" and "5422" platforms.
