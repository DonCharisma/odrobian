Introduction

ORGA Entertainment System is currently available for "s805" and "5422" platforms supporting both Mali-X11/Fbdev drivers.

What is ORGA Entertainment System?

Odrobian RetroGaming Arcade (ORGA) is basically an entertainment system supporting hardware acceleration for Multimedia, Gaming, Browsing, etc.

  • Desktop X11 Mali-GPU drivers (most functional multimedia experience based on MATE with multitasking)
  • Direct Frame Buffer Mali-GPU drivers (highest GPU performance possible with best CPU efficiency)

I have prepared specifically for retro emulation and multimedia with KODI for Media Centers using EmulationStaton as the main front end, all based on the beloved software combination that is well known and was first introduced by RetroPie with these formulas:

ORGA-X11 = (Mali-GPU-X11 + GLES-X11) * (RetroArch-X11 + KODI-X11 + EmulationStation-X11) / (Dedicated X11 GUI)

ORGA-Fbdev = (Mali-GPU-Fbdev + SDL2-Fbdev) * (RetroArch-Fbdev + KODI-Fbdev + EmulationStation-Fbdev) / (Handy X11 GUI)

ORGA for X11
  • Easy-to-use and very user friendly.
  • Will not affect MATE desktop nature.
  • Lots of customization.
  • Universal (currently works on all ODROIDs perfectly).
ORGA for Fbdev
  • Maximum performance.
  • Built specifically to benefit the most out of Retrogaming.
  • Not working as expected on "5422" platforms.

Getting Started

This is originally based on our main (OJ - MATE) distribution with a simple command that will convert the whole thing into this ORGA entertainment system, so first step is downloading ODROBIAN Jessie image for your current ODROID platform, you actually need to download the MATE edition then, after flashing and booting successfully, just open up a terminal (Ctrl + Alt + T):


	odroid:$ sudo -s
	odroid:# apt-get update && apt-get install io-cli
	This will basically upgrade Odrobian Platform to pull latest "insta-ORGA" utility.

	odroid:# insta-ORGA --defaults
	This will convert your system to RetroGaming Arcade with default recommended settings for your particular ODROID platform.
	
	odroid:# reboot

Default recommended settings are mainly based on Mali-X11 so either this or customize your installation by following the instructions documented for the insta-ORGA utility. After that, wait till your ODROID reboot, I strongly recommend that you can upgrade the kernel now:


	odroid:$ sudo apt-get dist-upgrade

You should do this instantly after converting the image to receive new kernels which should have less issues, please note that you should already have the latest regular software & packages updated automatically during "insta-ORGA" command.

Auto-Start EmulationStation

To auto-start EmulationStation directly after boot process for both ORGA X11/Fbdev on all platforms just do this.


	odroid:$ sudo insta-ORGA --enable-autostart
One important note, I have included workaround for the annoying background console text by when choosing to autostart emulationstation for ORGA Fbdev, and included many optimizations as well for the ORGA X11 autostart scripts. In fact, You can also revert back to default manual setting whenever you wish!


	odroid:$ sudo insta-ORGA --disable-autostart

Default Login User

Always log with the default "odroid" user and not the root, or else you will end up breaking a lot of things if you you're not sure about what you're doing:


	Username: odroid
	Password: odroid

So, forget about the root user and keep using "sudo" for your permission needs.
