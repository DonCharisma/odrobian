Start Gaming on ORGA

You should have your emulation system directory at your main home folder "/home/odroid/RetroGaming/" which is where you will put your stuff. The GUI will start automatically for you on ORGA for X11 however, If you are using ORGA for Fbdev with disabled GUI, you can switch to TTY3 (Ctrl + Alt + F3) and then temporarily use X11 to arrange and configure your stuff:


	odroid:$ startx

Prepare Your ROMs & BIOS files

If you have a BIOS file for PCSX (PS1) emulation as an example or anything else then put them directly inside the "~/RetroGaming/BIOS/" folder and they will be automatically detected, when it comes to game ROMs, we currently have these emulators located at "~/RetroGaming/roms" by default running as RetroArch cores, although we will be adding more very soon.

Run EmulationStation

Here's how you can run EmulationStation manually if you haven't already set it up to start automatically.

ORGA for X11

Navigate to ( Menua > All Applications > Games > EmulationStation ) as simple as this.

ORGA for Fbdev

As I told you before, currently we are on the console that I specified for X11 which is TTY3, you can switch back to TTY1 anytime you like to start your emulationstation front end, so just click (Ctrl + Alt + F1) and run this command:


	odroid:$ emulationstation

After adding roms to the default folder of your desired emulator as shown the previous step, it should automatically show up on your screen however, KODI media center is already pre-configured for you by default if you are not planning to play any games yet, please note that EmulationStation will first ask you to configure your joy-pad (you can use your attached keyboard if you want) for those of you aiming at multimedia usage.
