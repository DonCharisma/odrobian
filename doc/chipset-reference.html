ODROID Chipset Reference

Everything related to a specific ODROID platform on ODROBIAN repository is referred to by its chipset, so that's how you should recognize ODROIDs here.

Exynos 5422

ODROID XU3/XU4 (SOC) = "5422"

  • ARMv7 Architecture
  • 4 x ARM Cortex-A15 Quad 2.0GHz / 4 x Cortex-A7 Quad 1.4GHz
  • 6 x ARM Mali-T628
  • 2GB 32bit LPDDR3 RAM PoP 750Mhz
  • 10/100/1000Mbps Ethernet

AMLogic s805

ODROID C1/C1+/C0 (SOC) = "s805"

  • ARMv7 Architecture @28nm
  • 4 x ARM Cortex-A5 1.5GHz
  • 2 x ARM Mali-450 600MHz
  • 1GB 32bit DDR3 792MHz
  • 10/100/1000Mbps Ethernet

AMLogic s905

ODROID C2 (SOC) = "s905"

  • ARMv8 Architecture
  • 4 x ARM Cortex-A53 2Ghz
  • 3 x ARM Mali-450 600MHz
  • 2GB DDR3 SDRAM 792MHz
  • 10/100/1000Mbps Ethernet
