ODROBIAN 5422 (Stock Platform Material)

These files are already included by default on a stock ODROBIAN image however, I am stating these as a reference if you wish to build it from scratch or any other reason.

Platform Files Default Locations

These files should be put at:

  • /$BOOT/boot.ini
  • /etc/fstab
  • /etc/udev/rules.d/10-odroid.rules
  • /etc/X11/xorg.conf
  • /etc/init.d/zram
  • /etc/init.d/ondemand
  • /etc/asound.conf
  • /etc/modprobe.d/blacklist-odroid.conf

$BOOT = Boot Partition
